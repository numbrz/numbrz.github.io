Excel Spreadsheets

Numbrz supports Excel 365 Spreadsheets that conform to the following:

  • The data must begin at cell A1.
  • The cells must be in the form of a contiguous table.
  • The first row must be column labels (a header row).
  • So we can detect data types, there should be at least one data row.

Connecting to Excel 365

  • Locate the "Share URL" from the Excel 365 UI and copy it.
  • Paste the URL at the prompt.
  • Be ready to provide your Microsoft 365 user name and password to authorize Numbrz to access your data.

Configuring the Connection

Overview section
This section allows you to set a name and description associated with this data connection. In addition, should your authorization change or expire, you can click the Reconnect button to re-authorize access to your data. In the event of a reconnection, all current configuration, including connected data, is preserved.

Available Tables
This section shows the Spreadsheet tabs that are inside the Excel spreadsheet you linked, but not yet connected to the Numbrz application. To connect an item, select it from the list and click the Connect Table button. Upon successful connection, it will now appear in the Connected Tables section. Note that information about the available data is stored for short periods of time within Numbrz, if the desired item does not appear in the list, click the Reload button to refresh this information from the spreadsheet.

Connected Tables
This section shows the data you have connected and made available to the Numbrz application. If there are no items listed here, click the Available Tables section to select data. To manage a connected item, select it from the list. You can perform the following:

  • Edit the Name and Description of the item.
  • Refresh the columns from the source data. If columns appear in the source data, but are not in the description here, this will update the stored column information from the source.
  • Set the data Type of columns. Numbrz does its best to guess the best data type in most cases, but in the event that guess is incorrect, the type can be manually set.
  • Disconnect the item if it is no longer needed.
