Available Tables

This section shows data that is available, but not yet connected, to use within the Numbrz application. The items displayed here may vary, depending upon the data source. For example, connected spreadsheets will show the various sheet tabs in this section. To connect an item, and thus make it available to the Numbrz application, select it from the list and click the Connect... button. Upon successful connection, it will now appear in the Connected Tab.

Note that information about the available data is stored for short periods of time within Numbrz, if the desired item does not appear in the list, click the Reload button to refresh this information from the data source.
