Dropbox Files

Numbrz supports CSV (comma separated values) files located on Dropbox. Files must conform to the following:

  • The data must standards-compliant CSV.
  • For best results, place the files you wish to access in a dedicated folder.
  • The first row must be column labels (a header row).
  • So we can detect data types, there should be at least one data row.

Connecting to Dropbox

  • Type the folder name containing your files at the prompt - don't forget the leading slash ('/').
  • Be ready to provide your Dropbox user name and password to authorize Numbrz to access your data.
