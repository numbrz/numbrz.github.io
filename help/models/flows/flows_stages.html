Flow Stages

A Flow Stage processes sets of records by passing them through a sequence of Steps, which configure various transformations and calculations, including executing Functions. A sequence of stages can be related by the sharing of data. Data is shared, for example, when data written by one stage is used as input to another in the same Flow.

Create a New Stage

To create a new Stage, select Flow Stages from the right side bar, then click the New Stage button. You will be prompted for a name. Once created, you can format the input data and begin adding Steps.
