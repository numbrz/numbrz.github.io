Numbrz Jobs

A Job specifies how to execute a Flow by mapping connected Data (as configured by the user) that will:

  • Provide the input data fields required by the flow.
  • Receive the output data fields emitted by the flow.

The job is the executable unit of a Numbrz Model, running a job starts an analysis. Status information is provided as the job runs.

Create a New Job

To create a new Job, click the + icon in the lower right corner. You will be prompted to select the Flow that will provide the logic for this job, as well as a Name for the job. There are two options for mapping Data to the Flow.

Automatically Configure Mappings
Choose this option when you have already connected data that closely matches the inputs and outputs declared in the flow. Using data field name and type matching, Numbrz will attempt to create the needed data mappings.

Manually Configure Mappings
If you are unsure of what data is required by the flow, or do not know what data connections are already available, you may select this option. The automated mapping will be skipped and the job editor will be displayed so you can continue the configuration. Configuring a Job will explain this in detail.
