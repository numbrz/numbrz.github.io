Job Status

When a job is run, the Job Status screen is displayed.

While a job is running...

Running jobs emit information about job progress, including the number of records processed and the time taken to perform operations overall and per Flow Stage. Use the Cancel Job if you need to cancel a running job. If you navigate away from a job's status, you can always return to it via the Recent Runs section of the Model.

When a job ends...

Jobs end when they complete successfully, when an error occurs, or when cancelled by a user. The status will show the final result of the job. Two additional buttons are displayed when a job is completed. Use the Next Job button to select and run another job without having to return to the job list. Use the Rerun Job button to run the current job again.

See also: Run a Job, Recent Runs
